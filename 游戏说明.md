# 双人触摸屏格斗游戏

## 🎮 游戏简介
这是一个类似拳皇风格的双人同屏格斗游戏，专为触摸屏设备优化，支持两个玩家在同一台电脑上进行对战。

## 🚀 快速开始
双击 `index.html` 文件

## 🕹️ 操作方式

### 🆕 按键协同功能
游戏支持真正的按键协同：
- **玩家间独立**：两个玩家可以同时操作各自的控制器，互不影响
- **动作优先级**：单个玩家不能同时执行冲突动作（攻击 > 防御 > 移动）
- **多点触控**：触摸屏支持两个玩家同时操作各自的控制区域

### 触摸屏操作（推荐）
**玩家1（屏幕左侧）：**
- 方向键：移动和跳跃
- 拳：基础攻击
- 脚：强力攻击  
- 防：防御减少伤害

**玩家2（屏幕右侧）：**
- 方向键：移动和跳跃
- 拳：基础攻击
- 脚：强力攻击
- 防：防御减少伤害

### 键盘操作（测试用）
**玩家1：**
- A/D：左右移动
- W：跳跃
- Q：拳攻击
- E：脚攻击
- S：防御

**玩家2：**
- ←/→：左右移动
- ↑：跳跃
- 小键盘1：拳攻击
- 小键盘2：脚攻击
- 小键盘3：防御

## 🎯 游戏规则
- 每个玩家初始生命值：100
- 拳攻击伤害：10点
- 脚攻击伤害：15点
- 防御可减少50%伤害
- 生命值归零的玩家败北
- 支持空中战斗和连击

## 🎨 游戏特色
- **触摸屏优化**：大按钮设计，支持多点触控
- **流畅动画**：角色动作和特效
- **视觉效果**：攻击特效、受击效果、粒子效果
- **背景系统**：动态云朵和地面装饰
- **音效系统**：基础音效反馈
- **响应式设计**：适配不同屏幕尺寸

## 🔧 技术特性
- HTML5 Canvas 渲染
- 原生 JavaScript 游戏引擎
- 触摸屏和鼠标事件处理
- 60FPS 游戏循环
- 碰撞检测系统
- 状态管理系统

## 🐛 故障排除
1. **游戏无法加载**：确保使用现代浏览器（Chrome、Firefox、Edge）
2. **触摸屏无响应**：检查浏览器是否允许触摸事件
3. **性能问题**：关闭其他占用资源的程序
4. **音频无声音**：检查浏览器音频权限设置
